# LASTQUESTION FOREX

Premium website for LASTQUESTION FOREX — Forex & Crypto trading education community since 2021.

## Stack
- Next.js (App Router) + TypeScript
- TailwindCSS
- Framer Motion + GSAP
- Lucide React

## Getting Started
```bash
npm install
npm run dev
```

## Deploy
Deployed via Vercel, connected to this GitHub repository.
